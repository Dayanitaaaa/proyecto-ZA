# Proyecto ZA - Zapatillas Tejidas

Plataforma de e-commerce para la venta de zapatillas tejidas a mano.

## Características
- Catálogo de productos
- Sistema de carrito de compras
- Gestión de pedidos
- Panel de administración
- Responsive design

## Instalación
1. Activar entorno virtual: `source .venv/bin/activate`
2. Instalar dependencias: `pip install -r requirements.txt`
3. Migrar base de datos: `python manage.py migrate`
4. Crear superusuario: `python manage.py createsuperuser`
5. Iniciar servidor: `python manage.py runserver`

## Tecnologías
- Django 4.2.28
- Bootstrap 5
- SQLite
